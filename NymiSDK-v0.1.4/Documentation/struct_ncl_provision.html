NclProvision Struct Reference
Negotiation Types

Struct to store information about a provision (bridge between an NEA and a Nymi, used for validation the future).

Data Fields

NclProvisionKey key
 Secret key, used for secure communication between Nymi and NEA. Do not transfer this to other devices and NEAs.
 
NclProvisionId id
 The Nymi's name for this NEA, it also plays part in deriving the key. The ID doesn't need to be kept secret.

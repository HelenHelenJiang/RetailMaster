Detailed Description

A type of data-containing member of NclEvent. Say we received an NclEvent called event:

  • If event.type is NCL_EVENT_VK, the NclEventVk event.vk can be accessed.
  • If event.type is NCL_EVENT_GLOBAL_VK, the NclEventVk event.globalVk can be accessed.

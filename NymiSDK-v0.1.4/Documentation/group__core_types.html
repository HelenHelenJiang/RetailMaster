enum NclMode
Enumerator
NCL_MODE_DEFAULT 

Default run mode, with none of the following options enabled.

NCL_MODE_SYNCH 

Callbacks will be called from inside nclUpdate, which must be called regularly. If not set, callbacks happen when an event occurrs, possibly on a different thread.

NCL_MODE_DEV 

Interact with dev mode Nymis, not normal Nymis. Dev mode Nymis are always authenticated.

enum NclEventType
Enumerator
NCL_EVENT_ANY 

A special value used in nclAddBehavior and nclRemoveBehavior to indicate any event.

NCL_EVENT_INIT 

Library initialization has finished. Triggered by nclInit. Associated data will be stored in NclEvent::init.

NCL_EVENT_ERROR 

Something has failed. No associated data inside the event, but the error code will have been set. See nclGetErrorCode.

NCL_EVENT_DISCOVERY 

A provisioning Nymi has been discovered. Enabled by nclStartDiscovery. Associated data will be stored in NclEvent::discovery.

NCL_EVENT_FIND 

A specific provisioned Nymi has been found. Enabled by nclStartFinding. Associated data will be stored in NclEvent::find.

NCL_EVENT_DETECTION 

A Nymi has been detected. Enabled by nclStartFinding. Associated data will be stored in NclEvent::detection.

NCL_EVENT_AGREEMENT 

A shared secret has been agreed on by the Nymi and the NEA, and an LED pattern can now be shown to the user to ensure that the correct Nymi is being provisioned. Triggered by nclAgree. Associated data will be stored in NclEvent::agreement.

NCL_EVENT_PROVISION 

A Nymi has been provisioned. Triggered by nclProvision. Associated data will be stored in NclEvent::provision.

NCL_EVENT_VALIDATION 

A Nymi has been validated. Triggered by nclValidate. Associated data will be stored in NclEvent::validation.

NCL_EVENT_DISCONNECTION 

A Nymi's connection has terminated. Associated data will be stored in NclEvent::disconnection.

NCL_EVENT_ECG_START 

Finished starting the ECG stream of a Nymi. Triggered by nclStartEcgStream. Associated data will be stored in NclEvent::ecgStart.

NCL_EVENT_ECG 

A Nymi has sent ECG data. Triggered by nclStartEcgStream. Associated data will be stored in NclEvent::ecg.

NCL_EVENT_ECG_STOP 

Finished stopping the ECG of a Nymi. Triggered by nclStopEcgStream. Associated data will be stored in NclEvent::ecgStop.

NCL_EVENT_ACCEL_START 

Finished starting accelerometer stream of a Nymi. Triggered by nclStartAccelStream. Associated data will be stored in NclEvent::accelStart.

NCL_EVENT_ACCEL 

A Nymi has sent accelerometer data. Triggered by nclStartAccelStream. Associated data will be stored in NclEvent::accel.

NCL_EVENT_ACCEL_STOP 

Finished stopping accelerometer stream of a Nymi. Triggered by nclStopAccelStream. Associated data will be stored in NclEvent::accelStop.

NCL_EVENT_GYRO_START 

Finished starting gyroscope stream of a Nymi. Triggered by nclStartGyroStream. Associated data will be stored in NclEvent::gyroStart.

NCL_EVENT_GYRO 

A Nymi has sent gyroscope data. Triggered by nclStartGyroStream. Associated data will be stored in NclEvent::gyro.

NCL_EVENT_GYRO_STOP 

Finished stopping gyroscope stream of a Nymi. Triggered by nclStopGyroStream. Associated data will be stored in NclEvent::gyroStop.

NCL_EVENT_GESTURE_START 

Finished starting gesture stream of a Nymi. Triggered by nclStartGestureStream. Associated data will be stored in NclEvent::gestureStart.

NCL_EVENT_GESTURE 

A Nymi has recognized and sent a gesture. Triggered by nclStartGestureStream. Associated data will be stored in NclEvent::gesture.

NCL_EVENT_GESTURE_STOP 

Finished stopping gesture stream of a Nymi. Triggered by nclStopGestureStream. Associated data will be stored in NclEvent::gestureStop.

NCL_EVENT_VK 

A Nymi has sent a verification key. Triggered by nclCreateSigKeyPair. Associated data will be stored in NclEvent::vk.

NCL_EVENT_SIG 

A Nymi has signed a message and sent the signature. Triggered by nclSign. Associated data will be stored in NclEvent::sig.

NCL_EVENT_GLOBAL_VK 

A Nymi has sent a global verification key. Triggered by nclCreateGlobalSigKeyPair. Associated data will be stored in NclEvent::globalVk.

NCL_EVENT_GLOBAL_SIG 

A Nymi has signed a message with a global signing key and sent the signature. Triggered by nclGlobalSign. Associated data will be stored in NclEvent::globalSig.

NCL_EVENT_CREATED_SK 

A Nymi created and sent a symmetric key. Triggered by nclCreateSk. Associated data will be stored in NclEvent::createdSk.

NCL_EVENT_GOT_SK 

A Nymi sent a previously created symmetric key. Triggered by nclGetSk. Associated data will be stored in NclEvent::gotSk.

NCL_EVENT_PRG 

A Nymi sent a pseudorandom value. Triggered by nclPrg. Associated data will be stored in NclEvent::prg.

NCL_EVENT_RSSI 

A Nymi has sent its RSSI. Triggered by nclGetRssi. Associated data will be stored in NclEvent::rssi.

NCL_EVENT_FIRMWARE_VERSION 

A Nymi has sent its firmware version. Triggered by nclGetFirmwareVersion. Associated data will be stored in NclEvent::firmwareVersion.

NCL_EVENT_NOTIFIED 

Finished sending a Nymi the notify command. Triggered by nclNotify. Associated data will be stored in NclEvent::notified.

enum NclErrorCode
Enumerator
NCL_ERROR_NULL 

No error, or no appropriate programmatic response.

NCL_ERROR_NOT_INITED 

NCL has not been initialized. Call nclInit.

NCL_ERROR_NCL_FAILED 

NCL has failed irrecoverably. Do a postmortem if needed and call nclFinish.

NCL_ERROR_INVALID_HANDLE 

A Nymi handle specified doesn't correspond to an existing Nymi.

NCL_ERROR_MISMATCH 

The Nymi can't interpret what the NCL sent it or vice versa. Ensure the Nymi and NCL are compatible.

NCL_ERROR_WRONG_STATE 

A Nymi was in an incorrect state. For example, a provision was requested while validated.

NCL_ERROR_BUSY 

A Nymi was busy so another procedure couldn't be done. Leave the Nymi alone, wait, and try again.

NCL_ERROR_BAD_VALUE 

At least one parameter is out of range or doesn't specify a known entity.

NCL_ERROR_BAD_PARTNER_KEY 

The particular parameter partner key is invalid. Ensure the key is being specified correctly. If it is, check with the key distributor that the key is still valid. If it isn't, get a new one.

NCL_ERROR_OUT_OF_MEMORY 

The Nymi doesn't have enough memory for the requested action. Ask the user to clear out some memory.

NCL_ERROR_LOW_BATTERY 

Not enough battery to complete the command. Ask the user to charge the Nymi.

NCL_ERROR_NYMI_FAILED 

Catch-all for other failures on the Nymi that merit a retry. If it continues to fail, the Nymi may be permanently damaged.

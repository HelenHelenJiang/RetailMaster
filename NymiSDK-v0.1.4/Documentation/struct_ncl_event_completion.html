int nymiHandle
 The handle of the Nymi that completed a procedure.
 

Detailed Description

A type of data-containing member of NclEvent. Say we received an NclEvent called event:

  • If event.type is NCL_EVENT_VALIDATION, the NclEventCompletion event.validation can be accessed.
  • If event.type is NCL_EVENT_ECG_START, the NclEventCompletion event.ecgStart can be accessed.
  • If event.type is NCL_EVENT_ECG_STOP, the NclEventCompletion event.ecgStop can be accessed.
  • If event.type is NCL_EVENT_ACCEL_START, the NclEventCompletion event.accelStart can be accessed.
  • If event.type is NCL_EVENT_ACCEL_STOP, the NclEventCompletion event.accelStop can be accessed.
  • If event.type is NCL_EVENT_GYRO_START, the NclEventCompletion event.gyroStop can be accessed.
  • If event.type is NCL_EVENT_GYRO_STOP, the NclEventCompletion event.gyroStart can be accessed.
  • If event.type is NCL_EVENT_GESTURE_START, the NclEventCompletion event.gestureStart can be accessed.
  • If event.type is NCL_EVENT_GESTURE_STOP, the NclEventCompletion event.gestureStop can be accessed.
  • If event.type is NCL_EVENT_NOTIFIED, the NclEventCompletion event.notified can be accessed.
